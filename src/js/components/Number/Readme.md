### Number format

```jsx
<NumberFormat
  value="22"
/>
```

### Number format with prefix

```jsx
<NumberFormat
  value="22"
  prefix="$"
/>
```

### Number format with prefix and suffix

```jsx
<NumberFormat
  value="22"
  prefix="$"
  suffix="MXN"
/>
```

### Number format with prefix, suffix, and decimals

```jsx
<NumberFormat
  value="22"
  prefix="$"
  suffix="MXN"
  decimals="2"
/>
```
